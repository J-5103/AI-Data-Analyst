# 📊 AI Data Analyst Agent

An intelligent data analysis agent built with **LangGraph** and **LangChain** that automatically analyzes CSV data, generates visualizations, and produces business reports using free AI (Groq LLM).

📝 Node 4: Generating report...

## ✨ Features

- **Auto Data Loading** — Reads any CSV/Excel file automatically
- **AI Powered Analysis** — Ask questions in natural language
- **4 Chart Dashboard** — Auto generates Revenue, Sales, Trend, Units charts
- **Business Report** — Complete professional report generated by AI
- **100% Free** — Uses Groq API (no OpenAI needed!)
- **Multi-Node Workflow** — LangGraph manages entire pipeline

## 🛠️ Tech Stack

 **LangGraph** -> Multi-node workflow orchestration 
 **LangChain** -> Pandas Agent for data analysis |
 **Groq API** -> Free LLM (Llama 3.3 70B) |
 **Pandas** -> Data manipulation |
 **Matplotlib** -> Charts and visualizations |
 **openpyxl** -> Excel file support |
 **Python 3.10+** -> Programming language |

## 💡 Concepts Used

- **LangGraph** — StateGraph with multi-node workflow
- **LangChain** — Pandas DataFrame Agent
- **State Management** — TypedDict shared state
- **Tool Calling** — PythonREPL tool
- **Prompt Engineering** — Structured prompts for analysis
